How NGP's Discord-Server-Giveaways will look, and how moderators will follow, this documentation will go over most of the stuff that can happen.

Overview

This documentation is about how NGP Discord Server Giveaways work, and how everything will happen. We wil go over everything from How moderators will start giveaways to how long they will take to deliver the prize to the winner.

Before we continue please know that Moderators have the right to not give you your prize even after winning it, as long as they have a good reason that gets pass Flame, although we will try to make sure your prize gets to you.

Getting Started (moderators)

Below is how moderators can start giveaways using our giveaway bot made by NGP

Moderators can use the NGP Giveaway bot called "Better Giveaways" by doing the following:

  • Going to Discord
  • Going to the giveaway channel in a Discord server that they have admin/moderator in.
  • Typing in /bg_create along with their selected options for prize, duration, etc.

If Better Giveaways is still in beta, and admin testing only, you may have to ask someone with admin in the Discord Server, this acounts anyone with the @Founder Role.

After the giveaway ends this is what moderators do:

  1. The moderator that ran the giveaway must wait for the person who won the giveaway to claim. They must claim in the claim time - if there is any, if they do not claim in the claim time then the moderator can reroll
  2. Once the winner claimed, the moderator can do # CLAIMED exactly as put in the giveaways channel.
  3. The moderator has exactly 4 weeks to either decline the prize or give the prize too the winner. If they do not do this then the moderator will suffer consequences.
  4. If the winner does not claim then the moderator may reroll via /bg_reroll {msg_id}, they can get the msg id by right clicking the giveaway and clicking Copy Message ID
  5. The giveaway must follow NGP's ToS, otherwise the prize will be revoked.

Important Info:

If the moderator does not follow any of these steps in anyway/shape/form, then they will have consequences as stated before. These consequences may be:

  • Getting Fired from NGP
  • Being banned from being able to host giveaways in NGP.
  • Timeout in the server (if owner follows this documentation)
  • And any others based off Server Rules.

Getting Started (Members)

Members cannot host giveaways natively in NGP Studios. This changes if the owner allows members to host giveaways. But you members still have rules you must follow to help out the giveaway hosters.

Rules you must follow:

  1. Do not spam or post irrelevant content in giveaway channels.
  2. Do not attempt to manipulate or cheat the giveaway system.
  3. Follow all server rules and guidelines set by the server.
  4. Do not rush or pressure the giveaway hosters, otherwise you may lose your prize or be removed from the giveaway.

These rules are important to maintain a fair and enjoyable environment for all members in NGP. Even if the giveaway is not hosted in NGP Studios, members must still follow these rules to ensure fairness and respect for all participants. (unless the server owner allows otherwise)

/bg_edit id string number number Edits a giveaway.
/bg_list - - - - Lists all active giveaways in the server.

Common Issues

List to common problems and solutions:

Problem: Giveaway bot is offline

Solution: Try waiting for the bot to come back online, it may take a few hours or so. If any giveaways ended the bot will automatically cover it, and it will resume any giveaways.

Problem: The Giveaway bot isn't joining the server

Solution: Please contact us and we'll give you a proper invite link, and help you on fixing it.
